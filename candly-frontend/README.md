# Candly - Frontend

Frontend React/Vite application for Candly.

## Environment

Copy `.env.example` to `.env` and set the backend base URL:

```bash
cp .env.example .env
```

Example:

```env
VITE_API_BASE_URL=http://localhost:8000/api
```

## Run locally

```bash
cd candly-frontend
npm install
npm run dev
```
